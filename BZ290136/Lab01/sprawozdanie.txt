1. Zainstalowano klient git.
2. Po uruchomieniu konsoli skolonowano zadane repozytorium poleceniem:
	git clone https://github.com/InzynieriaOprogramowaniaAGH/MIFT2021.git - sklonowanie repozytorium 
3. Zapoznano się z instrukcją do laboratorium.
4. Użyto następujących poleceń celem wykonania podpounktu:
   git branch BZ290136 - utworzenie nowego brancha o nazwie "BZ290136"
   git checkout BZ290136 - przełaczenie sie na brancha o nazwie "BZ290136"
   mkdir BZ290136 - utworzenie folderu o nazwie "BZ290136"
   cd BZ290136 - zmiana folderu na folder BZ290136
   mkdir Lab01 - utworzenie folderu o nazwie "Lab01" 
   cd Lab01 - zmiana folderu na folder Lab01
   touch sprawozdanie.txt - utworzenie pliku tekstowego o nazwie "sprawozdanie.txt"
   git add sprawozdanie.txt - dodanie do aktualnego brancha pliku o nazwie "sprawozdanie.txt"
   git commit -m "add file sprawozdanie.txt" - wykonanie commita z zadaną wiadomością
   git add graf.png - dodanie do aktualnego brancha pliku o nazwie "graf.png"
   git commit -m "add file graf.png" - wykonanie commita z zadaną wiadomością
5. Dodano screenshoty z wykonywania zadań, wykonano commity i pushe, zmieniono nazwy screenshotów i ponownie wykonano commity i pushe.
6. Zmieniono na brancha na brancha "master":
   git checkout master
7. Wykonano merge'a brancha BZ290136 do brancha master i wykonano push.
