Zadanie 3
Komendy potrzebne do instalacji Dockera
Używane na podstawie instrukcji:

sudo apt-get update

sudo apt-get install \
>     apt-transport-https \
>     ca-certificates \
>     curl \
>     gnupg \
>     lsb-release

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

echo \
>   "deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu \
>   $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt-get update

sudo apt-get install docker-ce docker-ce-cli containerd.io

Weryfikacja instalacji:
sudo docker run hello-world

Zadanie 5
Utworzyłem plik Dockerfile
jego treść:

FROM ubuntu:latest

RUN apt-get update
RUN apt-get upgrade -y
RUN apt-get install -y git
    
Następnie zbudowałem obraz:
sudo docker build -f Dockerfile .

Zadanie 6
Aby dowiedzieć się jaki tag ma zbudowany obraz wylistowałem wszystkie dockerowe obrazy:

sudo docker images

Aby go uruchomić użyłem polecenia:
sudo docker run -it 1e0843b6f023 bash
