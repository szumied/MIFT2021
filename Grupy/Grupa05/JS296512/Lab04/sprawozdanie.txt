1. Stworzyłem brancha osobistego na te zajęcia (Grupa05-JS296512_Lab04) od mojego brancha grupowego.

1.1 Opublikowałem obraz z poprzednich zajęć w serwisie Docker HUB:

docker tag IMAGE_ID nazwa_uzytkownika/nazwa_repo
docker push nazwa_uzytkownika/nazwa_repo

2. Znalazłem komunikator na githubie:
https://github.com/binhxn/node-chat-app

3. Stworzyłem nowy obraz kontenera dockerowego i skonfigurowałem go tak, aby był w stanie wykonać operację budowania komunikatora:
sudo docker build -t docker-build:build . -f Dockerfile.build
sudo docker run -it -p 3000:3000 IMAGE_ID

4. Stwórzyłem nowy obraz kontenera dockerowego i skonfigurowałem go tak, aby był w stanie wykonać operację testowanie komunikatora korzystając ze zbudowanego wcześniej programu w pkt 3:
sudo docker build -t docker-test:test . -f Dockerfile.test

5. Stworzyłem plik docker-compose.yml. Stowrzyłem środowisko, które:
- ściąga z repo najnowszą wersję "komunikatora"
- buduje go
- testuje czy komunikator działa
sudo docker-compose up
